# Verkko-osoitteet ja laitteet - Tehtäväohjeet

Tässä dokumentissa kuvataan vaiheittaiset ohjeet, jotka auttavat määrittämään VyOS-reitittimet, switchit ja Lubuntu-työasemat. Tämä on tarkoitettu erityisesti tehtävään, jossa laitteiden tulee kommunikoida eri VLANeissa.

---

## 1. VyOS Router 1 Konfigurointi

VyOS Router 1 toimii gatewayna VLAN 5:lle.

1. Siirry VyOS Router 1:n konfiguraatiotilaan:
   ```bash
   configure
   ```

2. Aseta liitännät ja staattiset reitit:
   ```bash
   set interfaces ethernet eth0 address 192.168.15.1/26  # VLAN 5
   set protocols static route 192.168.15.64/26 next-hop 192.168.15.193  # Reitti Switch1:lle
   set protocols static route 192.168.15.128/26 next-hop 192.168.15.194  # Reitti Switch2:lle
   ```

3. Tallenna ja ota käyttöön:
   ```bash
   commit
   save
   exit
   ```

---

## 2. VyOS Router 2 Konfigurointi

VyOS Router 2 hallinnoi VLAN 10:tä.

1. Siirry VyOS Router 2:n konfiguraatiotilaan:
   ```bash
   configure
   ```

2. Aseta liitännät ja staattiset reitit:
   ```bash
   set interfaces ethernet eth0 address 192.168.15.65/26  # VLAN 10
   set protocols static route 192.168.15.0/26 next-hop 192.168.15.193  # Reitti Switch1:lle
   set protocols static route 192.168.15.128/26 next-hop 192.168.15.194  # Reitti Switch2:lle
   ```

3. Tallenna ja ota käyttöön:
   ```bash
   commit
   save
   exit
   ```

---

## 3. Switch 1 Konfigurointi

Switch 1 vastaa VLAN 5 -verkon liikenteestä.

1. Siirry Switch 1:n konfiguraatiotilaan ja poista oletus VLAN:
   ```bash
   configure terminal
   configure vlan default delete ports all
   ```

2. Lisää VLAN 5 ja määritä IP-osoite:
   ```bash
   create vlan "workstations"
   configure vlan workstations tag 5
   configure vlan workstations add ports 1, 2 tagged
   configure vlan workstations ipaddress 192.168.15.193 255.255.255.192
   ```

3. Tallenna:
   ```bash
   save
   ```

---

## 4. Switch 2 Konfigurointi

Switch 2 käsittelee VLAN 10:n liikenteen.

1. Poista oletus VLAN ja lisää VLAN 10:
   ```bash
   configure terminal
   configure vlan default delete ports all
   ```

2. Määritä VLAN 10 ja sen IP-osoite:
   ```bash
   create vlan "lazystations"
   configure vlan lazystations tag 10
   configure vlan lazystations add ports 1, 2 tagged
   configure vlan lazystations ipaddress 192.168.15.194 255.255.255.192
   ```

3. Tallenna:
   ```bash
   save
   ```

---

## 5. Lubuntu 1 (VLAN 5)

Lubuntu 1 määritetään VLAN 5:lle.

1. Avaa Lubuntu 1:n terminaali ja lisää IP-osoite:
   ```bash
   sudo ip addr add 192.168.15.2/26 dev eth0
   ```

2. Lisää oletusreitti:
   ```bash
   sudo ip route add default via 192.168.15.193
   ```

---

## 6. Lubuntu 2 (VLAN 10)

Lubuntu 2 määritetään VLAN 10:lle.

1. Avaa Lubuntu 2:n terminaali ja lisää IP-osoite:
   ```bash
   sudo ip addr add 192.168.15.66/26 dev eth0
   ```

2. Lisää oletusreitti:
   ```bash
   sudo ip route add default via 192.168.15.194
   ```

---

## 7. Testaus

1. VyOS Router 1:ltä pingaa Lubuntu 1:
   ```bash
   ping 192.168.15.2
   ```

2. VyOS Router 2:ltä pingaa Lubuntu 2:
   ```bash
   ping 192.168.15.66
   ```

3. Pingaa reitittimien välillä ja varmista, että VLANit ovat saavutettavissa:
   ```bash
   ping 192.168.15.1  # Router 1
   ping 192.168.15.65  # Router 2
   ```

---

## Huomioitavaa

- Varmista, että kaikki käytetyt laitteet tukevat määritettyjä VLANeja.
- Käytä annettuja IP-osoitteita ja VLAN-tageja tarkasti.
- Tarkista kytkentäkaapelit ja laiteasetukset, jos yhteydet eivät toimi odotetusti.
